感悟

  • 消息队列本质上，是把多个应用给连接了起来，相互之间做异步消息通讯用。连接的彼此，可能都是一套独立的生态系统，连接后，就产生了更为强大的软件程序。
  • LinkdIn的工程师讲过，消息队列的灵感，就是来源于Linux系统的管道设计。
  • 初级运维工程师，只会用简单的命令，高级的就知道用管道来连接多个命令，组合成更为复杂，能够完成更多任务的工具。队列也是一样的。
