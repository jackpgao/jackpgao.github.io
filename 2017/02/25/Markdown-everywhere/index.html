Markdown Everywhere

发表于 2017-02-25 | 分类于 效率 |

接触了markdown后，发现编辑文档是如此的有趣，不免会萌生出在哪都要用markdown的想法。

想想平时工作中，除了自己记录一些文档之外，邮件，也是一个重要战场，如果能在邮件里用上markdown，应该会很爽。

后记

搞这个需求，主要是为了满足用markdown语法写邮件的目的，诚然，我们可以在任何一个支持markdown的编辑好邮件，再c+v过去，但是对于一些简单的邮件，上面这种方式还是更加简单、友好一些。
